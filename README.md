# 高中学生身高情况统计分析系统

## 项目简介

本项目是一个基于 Python + Pandas + Matplotlib 的数据分析系统，用于统计和分析高中学生的身高情况。系统提供数据收集、统计分析、可视化展示和数据管理等功能。

## 技术栈

- **Python 3.7+**
- **Pandas** - 数据处理和分析
- **Matplotlib** - 数据可视化
- **NumPy** - 数值计算
- **SciPy** - 科学计算和统计分析

## 数据格式

### 学生数据字段

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| student_id | 字符串 | 学号 | 2023010101 |
| name | 字符串 | 姓名 | 张三 |
| gender | 字符串 | 性别（男/女） | 男 |
| grade | 字符串 | 年级（高一/高二/高三） | 高一 |
| class | 整数 | 班级 | 1 |
| height | 浮点数 | 身高(cm) | 175.5 |
| measure_date | 字符串 | 测量日期 | 2024-03-15 |

### 导入模板示例

```csv
student_id,name,gender,grade,class,height,measure_date
2023010101,张三,男,高一,1,175.5,2024-03-15
2023010102,李四,女,高一,1,162.0,2024-03-15
```

## 数据校验规则

- 学号：必填，不可重复
- 姓名：必填
- 性别：必须是"男"或"女"
- 年级：必须是"高一"、"高二"或"高三"
- 班级：1-50之间的整数
- 身高：140cm - 200cm 之间
- 测量日期：YYYY-MM-DD 格式

## 统计指标说明

- **平均值**：所有学生身高的算术平均
- **中位数**：排序后位于中间的值
- **众数**：出现频率最高的身高值
- **标准差**：身高数据的离散程度
- **四分位数**：Q1(25%)、Q3(75%)
- **IQR**：四分位距 (Q3-Q1)

## 可视化图表说明

1. **直方图**：展示身高分布的整体形态
2. **箱线图**：对比男女生身高的分布特征
3. **趋势图**：展示不同年级的平均身高变化
4. **饼图**：展示各身高区间的比例
5. **热力图**：展示性别和年级的交叉分析
6. **对比图**：与全国青少年平均身高对比

## 快速开始

1. 安装依赖
```bash
pip install -r requirements.txt
```

2. 运行程序
```bash
python main.py
```

3. 初始化示例数据（可选）
   - 选择菜单 8 -> 3 生成示例数据

4. 进行数据分析
   - 选择菜单 3 查看统计分析
   - 选择菜单 4 生成可视化图表
   - 选择菜单 7 生成完整报告

## 注意事项

1. 首次运行时会自动创建 `data` 和 `reports` 目录
2. 数据文件默认保存在 `data/students.csv`
3. 图表和报告默认保存在 `reports/` 目录
4. 建议定期备份数据（菜单 6 -> 4）
